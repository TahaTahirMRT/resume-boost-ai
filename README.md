# 🚀 ResumeBoost AI

Advanced ATS Resume Analyzer & LinkedIn Optimizer  
✨ Designed by Taha Tahir

---

## 📌 Overview

ResumeBoost AI is a browser-based web application that analyzes resumes against specific job roles and provides:

- ✅ ATS Match Score
- ✅ Keyword Gap Detection
- ✅ Smart Resume Feedback
- ✅ LinkedIn Headline Suggestions
- ✅ Resume Quality Metrics

This project was built as part of my 7th Semester IT coursework and demonstrates front-end development, algorithmic thinking, and product-oriented design.

## 🎯 Features

### 🔎 Multi-Role Resume Analysis
Supports analysis for:
- Frontend Developer
- Backend Developer
- Full Stack Developer
- Data Analyst
- DevOps Engineer
- Cyber Security Analyst
- Mobile Developer
- UI/UX Designer

---

### 📊 Smart Scoring System

Resume scoring is calculated using:

- 50% Role-specific keyword match
- 20% Action verb detection
- 15% Resume length optimization
- 15% Contact & LinkedIn presence detection

---

### 💼 LinkedIn Headline Generator

Automatically suggests professional LinkedIn headlines based on selected job role.

## 🚀 How to Run Locally

1. Clone the repository
2. Open `index.html` in your browser
3. Paste resume text
4. Select job role
5. Click Analyze

No backend or installation required.
